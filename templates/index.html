Help & Documentation

How to find your network range?

Windows: Open Command Prompt and type ipconfig. Look for your IPv4 address. If it's 192.168.1.10, your network range is 192.168.1.0/24

Linux/Mac: Open Terminal and type ifconfig or ip addr.

What do the operating system indicators mean?

TTL ≤ 64: Linux, Unix, or macOS systems
TTL ≤ 128: Windows operating systems
TTL ≤ 255: Network routers or infrastructure devices
